# 多言語対応（i18n）設計

フロントエンドUIを日本語・英語・韓国語の3言語で提供。

> **❌ 未実装:** 計画中機能

---

## 対応言語

| 優先度 | 言語 | ロケール | 状態 |
|--------|------|----------|------|
| - | 日本語 | `ja` | 既存（デフォルト） |
| 1 | 英語 | `en` | Phase 1 |
| 2 | 韓国語 | `ko` | Phase 2 |

---

## 対象範囲

| 対象 | 含まれる |
|------|---------|
| フロントエンドUI | ✅ ボタン、ラベル、メニュー、メッセージ |
| バックエンドAPIメッセージ | ❌ |
| ユーザー生成コンテンツ | ❌ デッキ名等 |

---

## 技術選定

### ライブラリ: typesafe-i18n

| 選定理由 | 説明 |
|---------|------|
| 型安全性 | キー、パラメータがTypeScriptで型チェック |
| typo検出 | 存在しないキー参照でコンパイルエラー |
| 未翻訳検出 | 型レベルで不足を検出 |
| Vue 3対応 | Composition APIと統合 |
| 軽量 | バンドルサイズが小さい |

### 翻訳運用

Claude Code + 翻訳エージェント（外部サービス不要）

---

## ディレクトリ構造

```
frontend/src/i18n/
├── index.ts              # 設定・初期化
├── i18n-types.ts         # 型定義（自動生成）
├── formatters.ts         # カスタムフォーマッター
├── ja/index.ts           # 日本語（ベース）
├── en/index.ts           # 英語
└── ko/index.ts           # 韓国語
```

---

## キー命名規則

```
{ページ/機能}.{コンポーネント}.{要素}
```

例: `auth.login.title`, `dashboard.stats.winRate`

---

## 主要カテゴリ

| カテゴリ | 説明 |
|---------|------|
| `common` | 共通（保存、キャンセル、エラー等） |
