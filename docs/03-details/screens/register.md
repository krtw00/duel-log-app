# ユーザー登録

> Status: Active
> 最終更新: 2026-01-24

## 画面概要

| 項目 | 内容 |
|------|------|
| パス | `/register` |
| 目的 | 新規アカウントを作成する |
| アクセス権 | 不要（認証済みの場合はダッシュボードにリダイレクト） |
| レスポンシブ | 中央寄せカード |

---

## フォーム

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|:---:|--------------|
| ユーザー名 | テキスト | ✅ | 3文字以上 |
| メールアドレス | メール入力 | ✅ | メール形式 |
| パスワード | パスワード入力（表示切替可） | ✅ | 6文字以上 |
| パスワード確認 | パスワード入力（表示切替可） | ✅ | パスワードと一致 |

---

## ユーザー操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| 登録 | 登録ボタン | Supabase Authアカウント作成 |
| ログイン画面へ | リンククリック | `/login`へ遷移 |

---

## 状態

| 状態 | 条件 | 表示内容 |
|------|------|---------|
| ローディング | 登録処理中 | ボタンにスピナー |
| メール確認待ち | 確認メール送信後 | 案内メッセージ → ログインへ遷移 |
| 即ログイン | メール確認不要設定時 | ダッシュボードへ遷移 |
| エラー | 登録失敗 | エラーメッセージ表示 |
