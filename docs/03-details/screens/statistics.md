# 統計

> Status: Active
> 最終更新: 2026-01-24

## 画面概要

| 項目 | 内容 |
|------|------|
| パス | `/statistics` |
| 目的 | デッキ別勝率・相性・値推移を可視化する |
| アクセス権 | 認証必須 |
| レスポンシブ | モバイル: テーブル横スクロール、チャート縦積み |

---

## 表示データ

### デッキ別勝率テーブル

| 列 | 表示形式 |
|----|---------|
| デッキ名 | テキスト |
| 勝率 | パーセンテージ |
| 勝数 | 整数 |
| 敗数 | 整数 |
| 総対戦数 | 整数 |

### デッキ使用分布チャート

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| 相手デッキ分布 | 円グラフ（パーセンテージ付き） | duels集計 |

### 相性マトリクス

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| 行: 自デッキ / 列: 相手デッキ | 勝率% + 戦績 | duels集計 |

色分けルール:

| 勝率 | 色 |
|------|-----|
| 60%以上 | 緑 |
| 40〜60% | 黄 |
| 40%未満 | 赤 |
| データなし | 灰 |

### 推移チャート（RATE/DC時のみ）

| 軸 | 内容 |
|----|------|
| X軸 | 対戦番号（時系列順） |
| Y軸 | レート値 / DC値 |

---

## ユーザー操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| ゲームモード切替 | タブ選択（RANK/RATE/EVENT/DC） | 全統計をフィルタ |
| 年月選択 | ドロップダウン | 表示期間を変更 |
| 期間タイプ切替 | all / range トグル | フィルタ方式を変更 |
| 範囲指定 | 数値入力（開始〜終了） | 対戦番号でフィルタ |
| 自デッキフィルタ | ドロップダウン | 特定デッキの統計のみ表示 |
| フィルタリセット | リセットボタン | 全フィルタを初期値に戻す |

---

## 状態

| 状態 | 条件 | 表示内容 |
|------|------|---------|
| ローディング | データ取得中 | スピナー |
| 空 | 該当期間のデータなし | 空状態メッセージ |
| エラー | API通信失敗 | トースト通知 |
