# フィードバック

> Status: Active
> 最終更新: 2026-01-24

## 画面概要

| 項目 | 内容 |
|------|------|
| パス | `/feedback` |
| 目的 | バグ報告・機能要望・問い合わせを送信する |
| アクセス権 | 不要（未認証でもアクセス可） |
| レスポンシブ | 中央寄せフォーム（max-width 700px） |

---

## 表示データ

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| 紹介テキスト | カード内テキスト | i18n |
| SNSリンク | X(Twitter) + GitHub | 設定値 |
| GitHub無効警告 | アラート（GitHub連携未設定時） | 設定値 |

---

## タブ構成

| タブ | 目的 |
|------|------|
| バグ報告 | 不具合の報告 |
| 機能要望 | 改善提案の送信 |
| お問い合わせ | 一般的な連絡 |

---

## フォーム

### バグ報告タブ

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|:---:|--------------|
| タイトル | テキスト | ✅ | 空文字禁止 |
| 説明 | テキストエリア（4行） | ✅ | 空文字禁止 |
| 再現手順 | テキストエリア（3行） | - | - |
| 期待動作 | テキストエリア（2行） | - | - |
| 実際の動作 | テキストエリア（2行） | - | - |

### 機能要望タブ

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|:---:|--------------|
| タイトル | テキスト | ✅ | 空文字禁止 |
| 説明 | テキストエリア（4行） | ✅ | 空文字禁止 |
| ユースケース | テキストエリア（3行） | - | - |

### お問い合わせタブ

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|:---:|--------------|
| 件名 | テキスト | ✅ | 空文字禁止 |
| メッセージ | テキストエリア（6行） | ✅ | 空文字禁止 |

---

## ユーザー操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| タブ切替 | タブ選択 | フォーム表示を切替 |
| 送信 | 送信ボタン | GitHub Issue作成（連携有効時） |
| SNSリンク | リンククリック | 外部サイトを新タブで開く |

---

## 状態

| 状態 | 条件 | 表示内容 |
|------|------|---------|
| GitHub無効 | GitHub連携未設定 | 警告アラート + 送信ボタン無効 |
| 送信中 | フォーム送信中 | ボタンにスピナー |
| 送信完了 | 正常送信後 | 成功トースト通知 |
