# ログイン

> Status: Active
> 最終更新: 2026-01-24

## 画面概要

| 項目 | 内容 |
|------|------|
| パス | `/login` |
| 目的 | メール/パスワードまたはOAuthでログインする |
| アクセス権 | 不要（認証済みの場合はダッシュボードにリダイレクト） |
| レスポンシブ | デスクトップ: 分割レイアウト / モバイル: 全幅 |

---

## 表示データ

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| ロゴ | 画像 | 静的アセット |
| 言語切替 | セレクタ（右上） | i18n設定 |
| 配信者モードトグル | スイッチ（紫色） | localStorageまたはユーザー設定 |
| 利用規約 | モーダル（スクロール可能） | 静的テキスト |

---

## フォーム

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|:---:|--------------|
| メールアドレス | メール入力 | ✅ | メール形式 |
| パスワード | パスワード入力（表示切替可） | ✅ | 空文字禁止 |

---

## ユーザー操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| ログイン | ログインボタン | Supabase Auth認証 → ダッシュボード遷移 |
| Google OAuth | Googleボタン | Google OAuth認証フロー開始 |
| Discord OAuth | Discordボタン | Discord OAuth認証フロー開始 |
| 登録画面へ | 登録ボタン | `/register`へ遷移 |
| パスワード忘れ | リンククリック | `/forgot-password`へ遷移 |
| 言語切替 | セレクタ変更 | UIの表示言語を変更 |
| 配信者モード | トグル切替 | メールマスク表示を有効化 |
| 利用規約表示 | リンククリック | モーダル表示 |

---

## 状態

| 状態 | 条件 | 表示内容 |
|------|------|---------|
| ローディング | 認証処理中 | ボタンにスピナー |
| エラー | 認証失敗 | エラーメッセージ表示 |
| メール保存 | 前回ログイン時 | localStorage保存のメールを自動入力 |
