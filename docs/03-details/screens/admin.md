# 管理画面

> Status: Active
> 最終更新: 2026-01-24

## 画面概要

| 項目 | 内容 |
|------|------|
| パス | `/admin` |
| 目的 | システム全体の統計閲覧・ユーザー管理・メンテナンスを行う |
| アクセス権 | 管理者（is_admin=true）のみ |
| レスポンシブ | タブ内コンテンツが1カラムに縮小 |

---

## 表示データ

### タブ構成

| タブ | 内容 |
|------|------|
| ユーザー | ユーザー一覧・検索・管理操作 |
| 統計 | アプリ全体の利用統計チャート |
| メタ分析 | デッキメタ分析・マッチアップトレンド |
| メンテナンス | DB操作・キャッシュ管理 |

### ユーザー管理タブ

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| ユーザー一覧 | テーブル（名前・メール・登録日・ロール） | GET /admin/users |
| ユーザー検索 | テキスト入力でフィルタ | クライアントサイド |

### 統計タブ

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| 全体利用統計 | チャート | GET /admin/statistics |

### メタ分析タブ

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| デッキ使用率トレンド | チャート | GET /admin/meta-analysis |
| マッチアップ傾向 | テーブル | GET /admin/meta-analysis |

---

## ユーザー操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| ユーザー検索 | テキスト入力 | 一覧をフィルタ |
| ユーザー削除 | 削除ボタン | 確認ダイアログ後に削除 |
| ユーザーBAN | BANボタン | アカウント無効化 |
| BAN解除 | 解除ボタン | アカウント有効化 |
| タブ切替 | タブ選択 | 表示内容を切替 |

---

## 状態

| 状態 | 条件 | 表示内容 |
|------|------|---------|
| ローディング | データ取得中 | スピナー |
| 権限なし | is_admin=falseでアクセス | リダイレクト |
| エラー | API通信失敗 | トースト通知 |
