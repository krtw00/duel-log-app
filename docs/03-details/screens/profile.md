# プロフィール

> Status: Active
> 最終更新: 2026-01-24

## 画面概要

| 項目 | 内容 |
|------|------|
| パス | `/profile` |
| 目的 | ユーザー情報の変更・配信者モード設定・アカウント管理を行う |
| アクセス権 | 認証必須 |
| レスポンシブ | 中央寄せカード（max-width 600px） |

---

## 表示データ

| データ項目 | 表示形式 | データソース |
|-----------|---------|-------------|
| ユーザー名 | テキスト入力 | users.username |
| メールアドレス | テキスト入力（配信者モード時は読取専用） | users.email |
| 配信者モード | トグルスイッチ | users.streamer_mode |
| 画面解析 | トグルスイッチ（実験的機能バッジ付き） | users.screen_analysis |

---

## ユーザー操作

| 操作 | トリガー | 結果 |
|------|---------|------|
| プロフィール更新 | 保存ボタン | PUT /users/{id} |
| パスワード変更 | 新PW入力 + 保存 | Supabase Auth更新 |
| 配信者モード切替 | トグルスイッチ | メールマスク有効化 |
| 画面解析切替 | トグルスイッチ | 実験的機能の有効/無効 |
| データエクスポート | エクスポートボタン | 全データCSVダウンロード |
| データインポート | インポートボタン | .csvファイル選択・取込 |
| アカウント削除 | 削除ボタン | 確認ダイアログ表示 |

---

## フォーム

### プロフィール設定

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|:---:|--------------|
| ユーザー名 | テキスト | ✅ | 空文字禁止 |
| メールアドレス | メール | ✅ | メール形式 |
| 新パスワード | パスワード | - | 8-72文字（入力時のみ） |
| パスワード確認 | パスワード | - | 新パスワードと一致 |

### アカウント削除確認

| フィールド | 型 | 必須 | バリデーション |
|-----------|-----|:---:|--------------|
| 確認テキスト | テキスト | ✅ | 「DELETE」と完全一致 |

---

## 状態

| 状態 | 条件 | 表示内容 |
|------|------|---------|
| ローディング | 保存処理中 | ボタンにスピナー表示 |
| エラー | 更新失敗 | トースト通知 |
