# UI構想
## 戦績画面
- 表形式
- 入力と戦績を一体化
- 入力については以下
    - PC：インライン or サイドモーダル
    - モバイル：モーダル
```
┌────────────────────────────────────────────────────────────┐
│                        戦績一覧画面                        │
├────────────────────────────────────────────────────────────┤
│ フィルター： [デッキ選択] [勝敗] [期間] [ランク] [検索]             │
├────────────────────────────────────────────────────────────┤
│ セット | 対戦数 | 自分のデッキ | コイン | 先手/後手 | 相手のデッキ | 勝敗 |ランク| 登録日 | 備考 │
│───────┼────────┼────────────┼───────┼────────────┼──────────────┼──────┼─────────┼────────┼──────┤
│   1   |   1    | 烙印         | 表     | 先手       | 青眼         | 勝ち |   D1   | 9/24   | ○○○ │
│   1   |   2    | 烙印         | 裏     | 後手       | エルド        | 負け |  D1   | 9/23   | △△△ │
│   1   |   3    | 烙印         | 表     | 先手       | サンドラ      | 勝ち |  M1   |  9/22   | □□□ │
│  ...  |  ...   |              |        |            |               |      |        |      │
├────────────────────────────────────────────────────────────┤
│                                                  ＋        │
└────────────────────────────────────────────────────────────┘
```

```
┌────────────────────────────────────────────┐
│               新規対戦記録の入力           │
├────────────────────────────────────────────┤
│ 自分のデッキ： [選択]                      │
│ コイン：       [表/裏]                     │
│ 先手/後手：    [先/後]                      │
│ 相手のデッキ： [入力]                      │
│ 勝敗：         [勝ち/負け]
│ ランク：　　　　[B2～M1]                   |←ランクとレートは切り替え項目
| レート:        [数字]
│ 備考：         [テキストエリア]            │
│ 登録日：       [自動入力]          │
├────────────────────────────────────────────┤
│ [キャンセル]                     [保存]     │
└────────────────────────────────────────────┘
```
## 統計画面
```
┌────────────────────────────────────────────────────────────┐
│                        統計表示画面                        │
├────────────────────────────────────────────────────────────┤
│ 🔍 フィルター： [期間] [デッキ選択] [先手/後手] [勝敗]       │
├────────────────────────────────────────────────────────────┤
│ 🗂 タブ： [ 表形式 ] [ グラフ形式 ]                         │
├────────────────────────────────────────────────────────────┤

▼ 表形式タブ
┌────────────────────────────────────────────┐
│ 📋 概要統計                                 │
│ 対戦数：100   コイン勝率：50%   先行率：60% │
│ 勝率：60%     先攻勝率：63%     後攻勝率：40% │
└────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│ 📋 デッキ別統計                                              │
│ 使用デッキ     | 使用率 | 勝率 | 先攻勝率 | 後攻勝率         │
├────────────────────────────────────────────────────────────┤
│ 烙印           | 30%    | 60%  | 60%      | 40%              │
│ エルド         | 40%    | 50%  | 55%      | 35%              │
│ 白き森         | 20%    | 40%  | 50%      | 30%              │
└────────────────────────────────────────────────────────────┘

▼ グラフ形式タブ
┌────────────────────────────────────────────┐
│ 📊 対戦相手のデッキ比率（円グラフ）         │
│ 📊 相手デッキ別勝率（棒グラフ）             │
│ 📊 期間別勝率推移（折れ線グラフ）           │
└────────────────────────────────────────────┘
```
