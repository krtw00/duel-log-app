# UI構想
## 戦績画面
- 表形式
- 入力と戦績を一体化
- 入力については以下
    - PC：インライン or サイドモーダル
    - モバイル：モーダル
```
┌────────────────────────────────────────────────────────────┐
│                        戦績一覧画面                        │
├────────────────────────────────────────────────────────────┤
│ フィルター： [デッキ選択] [勝敗] [期間] [検索]             │
├────────────────────────────────────────────────────────────┤
│ セット | 対戦数 | 自分のデッキ | コイン | 先手/後手 | 相手のデッキ | 勝敗 | 登録日 | 備考 │
│───────┼────────┼────────────┼───────┼────────────┼──────────────┼──────┼────────┼──────┤
│   1   |   1    | 烙印         | 表     | 先手       | 青眼         | 勝ち | 9/24   | ○○○ │
│   1   |   2    | 烙印         | 裏     | 後手       | エルド        | 負け | 9/23   | △△△ │
│   1   |   3    | 烙印         | 表     | 先手       | サンドラ      | 勝ち | 9/22   | □□□ │
│  ...  |  ...   |              |        |            |               |      |        |      │
├────────────────────────────────────────────────────────────┤
│                                                  ＋        │
└────────────────────────────────────────────────────────────┘
```

```
┌────────────────────────────────────────────┐
│               新規対戦記録の入力           │
├────────────────────────────────────────────┤
│ 自分のデッキ： [選択]                      │
│ コイン：       [表/裏]                     │
│ 先手/後手：    [先/後]                      │
│ 相手のデッキ： [入力]                      │
│ 勝敗：         [勝ち/負け]                │
│ 備考：         [テキストエリア]            │
│ 登録日：       [自動入力]          │
├────────────────────────────────────────────┤
│ [キャンセル]                     [保存]     │
└────────────────────────────────────────────┘
```
