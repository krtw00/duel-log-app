# スコープ・対象外

> Status: Active
> 最終更新: 2026-01-23

Duel Log Appのスコープ（対象範囲）と対象外を明確に定義する。

---

## スコープ内（対象範囲）

| カテゴリ | 内容 |
|---------|------|
| ユーザー管理 | 登録、認証（OAuth）、プロフィール |
| 対戦記録 | 手動入力、CSVインポート、画面解析による自動入力 |
| デッキ管理 | 自分のデッキ・対戦相手デッキの登録・管理 |
| 統計表示 | 勝率、先後攻勝率、相性表、期間別統計 |
| データ共有 | 統計共有URL、CSVエクスポート |
| 配信者向け | ポップアップ表示、配信者モード（プライバシー保護） |
| 管理機能 | ユーザー管理、全体統計 |

---

## スコープ外（対象範囲外）

| カテゴリ | 理由 |
|---------|------|
| カード情報の管理 | カードデータベースは外部サービスが充実 |
| デッキ構築支援 | デッキレシピ管理は対象外 |
| 対戦マッチング | 対戦相手を探す機能は不要 |
| リアルタイム対戦 | ゲームプレイ機能は不要 |
| 課金機能 | 無料サービスとして提供 |
| ネイティブアプリ | Web（PWA可能）のみ |
| チャット・メッセージ | コミュニケーション機能は不要 |

---

## 対象ユーザー

| ユーザー | 利用シーン |
|---------|-----------|
| 一般プレイヤー | 対戦記録の入力、統計情報の確認 |
| 配信者 | ポップアップでリアルタイム統計表示、プライバシー保護 |
| 管理者 | ユーザー管理、データメンテナンス |

---

| 対象 | 備考 |
|------|------|
